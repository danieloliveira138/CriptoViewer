# CriptoViewer
Android App that displays coins from Coin Market Cap

## Configuração Necessária

Para utilizar este projeto, é necessário criar um arquivo chamado `secrets.properties` na raiz do projeto e adicionar a sua chave de API da CoinMarketCap da seguinte forma:

```properties
CMC_API_KEY=SUA_CHAVE_DE_API_AQUI
```
